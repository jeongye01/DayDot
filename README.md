# 🌟 DayDot — 매일의 점이 모여 빛나는 당신이 되는 곳

DayDot은 **하루의 감정과 한 줄 기록을 빠르게 남기는** 감정 일기 서비스입니다.

---

## ✨ Features (MVP)

### 📝 1. 하루 한 줄 기록

- 감정 5단계 선택 (Happy, Good, Neutral, Bad, Angry)
- 한 줄 기록(optional)

### 🔥 2. 리텐션 기능

- 연속 기록 일수 / 최장 연속 기록 일수 제공

### 📅 3. Dots (History)

- 캘린더에서 기록 목록 조회
  - 캘린더에서 기록을 선택해 단건 조회, 수정, 삭제 가능
- 월간 감정 통계를 라인 차트로 시각화
  - 감정별 색상 매핑

---

## 🏗 Tech Stack

### **Frontend**

- **Next.js 15** (App Router, Turbopack)
- **TypeScript**
- **Tailwind CSS**
- **shadcn-ui**
- **Recharts** (감정 차트)
- **Tanstack-Query**

### **Backend**

- **Next.js API Routes**
- **Prisma ORM**
- **PostgreSQL (Neon)**
